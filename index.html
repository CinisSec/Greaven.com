Re-organizing blogposts & daemons

08-04-2023

Hey there!

I have been re-organizing the website because of some flow changes. I've noticed some issues with the current way I was blogging stuff. I mean it was easy for me to blog about stuff because it was mostly some div blocks. I've decided to put all the blogposts in a subdirectory and keep the general posts on the home page. The blog page will be an index of articles.

I also wanted to share a bit about my adventures using FreeBSD. I recently decided to wake up my old laptop from 2011 from it's slumber to try out FreeBSD again. I remember trying it back in 2017-2018 but I was so lost coming from a mainly Windows and a slight Linux usage. I mean I was using Arch but I had no idea about what I was doing... chmod 777 anyone?

Anyway the installation of FreeBSD was pretty straight forward but I ended up not configuring my wifi properly. As it was a fresh install I just decided to install FreeBSD again and everything worked flawlessly! I must admit I quite enjoy the system and user file separation. I feel like Linux systems can become quite messy if not cared for properly.

Later that day I tried to mount an SMB fileshare on the FreeBSD system and... of course SMB2+ is not officially supported by FreeBSD. My plan now is to mount my NAS with NFS but that's a work in progress!

Have a nice day!
