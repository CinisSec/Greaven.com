Format and partition a drive on Linux Part 2

08-06-2023

As I've been using linux workstations more frequently I realized my previous blogpost was only applicable if you need to use the disk on other non linux systems.
If you only want to use the disk on Linux systems you can just do the following: mkfs.ext4 /dev/sdx -L mylabel Instead of partitioning the disk you'll just make a filesystem on sdx.
The next step would be to just mount sdx and there you go!
I hope this helped!
